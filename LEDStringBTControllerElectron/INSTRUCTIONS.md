

## Install dependencies:

```
npm install node-ble

```


Add "start": "electron ." to your package.json.

## Run app

```
npm install
npm start
```

## Build app
```
npm run build
```


## Release new app version

To trigger a release:

Commit your changes.
Push a new tag

```
git tag v1.1.0
git push origin v1.1.0
```
